\subsubsection{Detailed Description}
This is the abstract base class for all fields, wether they contain a floating-\/point parameter, or a string,... 

All \doxyref{W\+X\+Field}{p.}{a00125} have a title and an entry field.

Note that \doxyref{W\+X\+Field\+::\+Cryst\+Update()}{p.}{a00122_a7ac00ae2ae28f1a6fa26e6fa571186b6} and \doxyref{W\+X\+Field\+::\+Update\+U\+I()}{p.}{a00122_a3818940b7031ff7e45cf2178c4a99c90} should be done from the parent object. Notably using \doxyref{W\+X\+Field\+::\+Cryst\+Update}{p.}{a00122_a7ac00ae2ae28f1a6fa26e6fa571186b6}(updateui=true) will not trigger an update of the U\+I.
