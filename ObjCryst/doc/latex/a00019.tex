\subsubsection{Detailed Description}
The \doxyref{CIFData}{p.}{a00019} class holds all the information from a {\itshape single\/} data\_\- block from a cif file. It is a placeholder for all comments, item and loop data, as raw strings copied from a cif file.

It is also used to interpret this data to extract parts of the cif data, i.e. only part of the core cif dictionnary are recognized. \doxyref{CIF}{p.}{a00017} tags currently recognized include (\char`\"{}tag1 $>$ tag2\char`\"{} means tag1 is preferred to tag2 when extracting the info, only one is reported):
\begin{DoxyItemize}
\item crystal name: \_\-chemical\_\-name\_\-systematic $>$ \_\-chemical\_\-name\_\-mineral $>$ \_\-chemical\_\-name\_\-structure\_\-type $>$ \_\-chemical\_\-name\_\-common
\item crystal formula: \_\-chemical\_\-formula\_\-analytical $>$ \_\-chemical\_\-formula\_\-structural $>$ \_\-chemical\_\-formula\_\-iupac $>$ \_\-chemical\_\-formula\_\-moiety
\item unit cell: \_\-cell\_\-length\_\-\{a,b,c\} ; \_\-cell\_\-angle\_\-\{alpha,beta,gamma\}
\item spacegroup number: \_\-space\_\-group\_\-IT\_\-number $>$ \_\-symmetry\_\-Int\_\-Tables\_\-number
\item spacegroup Hall symbol: \_\-space\_\-group\_\-name\_\-Hall $>$ \_\-symmetry\_\-space\_\-group\_\-name\_\-Hall
\item spacegroup Hermann-\/Mauguin symbol:\_\-space\_\-group\_\-name\_\-H-\/M\_\-alt $>$ \_\-symmetry\_\-space\_\-group\_\-name\_\-H-\/M
\item atom coordinates: \_\-atom\_\-site\_\-fract\_\-\{x\} ; \_\-atom\_\-site\_\-Cartn\_\-\{x,y,z\}
\item atom occupancy: \_\-atom\_\-site\_\-occupancy
\item atom label \& symbol: \_\-atom\_\-site\_\-type\_\-symbol ; \_\-atom\_\-site\_\-label
\end{DoxyItemize}

Cartesian coordinates are stored in Angstroems, angles in radians.

To import \doxyref{PowderPattern}{p.}{a00061} data, the following tags are used:
\begin{DoxyItemize}
\item observed intensity: \_\-pd\_\-meas\_\-counts\_\-total $>$ \_\-pd\_\-meas\_\-intensity\_\-total $>$ \_\-pd\_\-proc\_\-intensity\_\-total $>$ \_\-pd\_\-proc\_\-intensity\_\-net
\item uncertainty on intensity: deducted from \_\-pd\_\-proc\_\-ls\_\-weight or square root of intensity
\item coordinates: \_\-pd\_\-proc\_\-2theta\_\-corrected $>$ \_\-pd\_\-meas\_\-angle\_\-2theta $>$ \_\-pd\_\-meas\_\-time\_\-of\_\-flight $>$ \_\-pd\_\-proc\_\-2theta\_\-range\_\-\{min,max,inc\}
\item intensity normalizer (optional): \_\-pd\_\-meas\_\-intensity\_\-monitor $>$ \_\-pd\_\-meas\_\-step\_\-count\_\-time
\end{DoxyItemize}

If another data field is needed, it is possible to directly access the string data (\doxyref{CIFData::mvComment}{p.}{a00019_a9408fb91caf78518c74b0de53a7c4d38} , \doxyref{CIFData::mvItem}{p.}{a00019_a50f7e91571f28b3a3a66675a564496a6} and \doxyref{CIFData::mvLoop}{p.}{a00019_aebd5e242b7771f184f29647527fcab5a}) to search for the correct tags.
