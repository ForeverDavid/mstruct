\subsubsection{Detailed Description}
Generic class for parameters of refinable objects. These must be continuous.

\begin{Desc}
\item[{\bf Todo}]: define parameters using equations between parameters. 

: for complex objects with lots of parameters, give the possibility to define vectors of parameters, all with the same properties, to reduce memory usage. \end{Desc}
