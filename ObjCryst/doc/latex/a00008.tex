\subsubsection{\-Detailed \-Description}
\-The basic atom scatterer, in a crystal. 

\-This class records the position of the atom, and has a pointer to its \doxyref{\-Scattering\-Power\-Atom}{p.}{a00091}.

\begin{DoxyNote}{\-Note}
there can be '\-Dummy' atoms, for which the used symbol is \char`\"{}\-X\char`\"{}, and which have no scattering power (use with caution\-: dummy atoms are only supposed to be used within \doxyref{\-Z\-Scatterer}{p.}{a00156}) 
\end{DoxyNote}
