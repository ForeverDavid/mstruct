\subsubsection{Detailed Description}
class of refinable parameter types. This is used to categorize all parameters, to be able to modify a complete category of parameters (fix/unfix, set limits,...). The types are organized as a tree.

Parameters should be allocated globally in the heap, so we only use pointers in the interface.

\begin{DoxyNote}{Note}
when modifying (fixing, changing limits) for a given \doxyref{RefParType}{p.}{a00079} (ie a 'family' of parameters), it affects all \doxyref{RefinablePar}{p.}{a00072} of this type {\itshape and\/} the parameters belonging to the children of this \doxyref{RefParType}{p.}{a00079}. eg fixing for the type \char`\"{}gpRefParTypeScatt\char`\"{} will fix all the derived postionnal, orientationnal, and population parameters for the scatterers.
\end{DoxyNote}
