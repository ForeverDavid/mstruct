\subsubsection{Detailed Description}
A field with the name of a \doxyref{WXCrystObj}{p.}{a00114}. Updating must be done by the \doxyref{WXCrystObj}{p.}{a00114} owner. For a simple string field linked directly to a string, use \doxyref{ObjCryst::WXFieldString}{p.}{a00125}

The idea is to only use a few Mutexes to lock data from the top objects (wxRefinableObj,...), when calling \doxyref{CrystUpdate()}{p.}{a00120_a538a45567f480e1716ca3b593a1d0ed2} and \doxyref{UpdateUI()}{p.}{a00120_a490d3cd6888b9fd6c59dabe031e17440}. As sub-\/objects (\doxyref{WXField}{p.}{a00118},...) are only updated from within a top object, the mutex lock in the top object will also lock the data in the sub-\/objects.
