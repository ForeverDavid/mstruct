\subsubsection{Detailed Description}
DiffractionData object for Single \doxyref{Crystal}{p.}{a00020} analysis. Currently this handles only in the simplest way single crystal dat: ie only data which has been completely corrected for Lorentz/Polarization and absorption.

What needs to be developped: define the geometry of the experiment (incident and emerging angles), the polarization of the beam, etc...
