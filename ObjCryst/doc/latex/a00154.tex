\subsubsection{Detailed Description}
\doxyref{ZPolyhedron}{p.}{a00154}: a \doxyref{Scatterer}{p.}{a00084} to describe polyhedras such as octahedron, tetrahedron, square plane, etc. .. These are \doxyref{ZScatterer}{p.}{a00155} objects, so that even if they are initialized with constraints, these can be removed to make any configuration.
