\subsubsection{Detailed Description}
Class to define the radiation (type, monochromaticity, wavelength(s)) of an experiment. This can be developped for more complex experiments, hence the {\itshape vector\/} of wavelengths (so far it is not possible to use several wavelengths, though).

X-\/Rays and Neutrons are available. Electrons are not available yet in \doxyref{ScatteringData}{p.}{a00088} classes.
