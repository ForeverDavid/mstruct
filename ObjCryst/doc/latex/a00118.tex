\subsubsection{Detailed Description}
This is the abstract base class for all fields, wether they contain a floating-\/point parameter, or a string,. .. All \doxyref{WXField}{p.}{a00118} have a title and an entry field.

Note that \doxyref{WXField::CrystUpdate()}{p.}{a00115_a7ac00ae2ae28f1a6fa26e6fa571186b6} and \doxyref{WXField::UpdateUI()}{p.}{a00115_a3818940b7031ff7e45cf2178c4a99c90} should be done from the parent object. Notably using \doxyref{WXField::CrystUpdate}{p.}{a00115_a7ac00ae2ae28f1a6fa26e6fa571186b6}(updateui=true) will not trigger an update of the UI.
