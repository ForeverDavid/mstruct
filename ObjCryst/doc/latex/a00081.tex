\subsubsection{Detailed Description}
Rigid groups of atoms inside a molecule. These atoms can be moved as a group using one rotation and one translation. \begin{DoxyWarning}{Warning}
: the rotation (quaternion) and translation parameters are not saved or displayed, so any time this must be done \doxyref{Molecule::ResetRigidGroupsPar()}{p.}{a00046_a9daa98a9b4a992d56e0dba41b104b291} {\bfseries must} be called beforehand -\/ this will use the rotation and translation parameters to generate the final atomic coordinates, and reset mQuat,mX,mY and mZ. 
\end{DoxyWarning}
