\subsubsection{Detailed Description}
Main \doxyref{CIF}{p.}{a00017} class -\/ parses the stream and separates data blocks, comments, items, loops. All values are stored as string, and Each \doxyref{CIF}{p.}{a00017} block is stored in a separate \doxyref{CIFData}{p.}{a00019} object. No interpretaion is made here -\/ this must be done from all \doxyref{CIFData}{p.}{a00019} objects.
