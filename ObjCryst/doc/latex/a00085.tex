\subsubsection{\-Detailed \-Description}
\-Generic type of scatterer\-: can be an atom, or a more complex assembly of atoms. 

\-A \doxyref{\-Scatterer}{p.}{a00085} is able to give its position (in fractionnal coordinates) in the unit cell, and more generally the position of all point scattering centers (\doxyref{\-Scattering\-Component}{p.}{a00086}), along with the \doxyref{\-Scattering\-Power}{p.}{a00090} associated with each position.

\-For simple atoms, there is only one scattering position with the associated scattering power (scattering factor, anomalous, thermic). \-For complex scatterers (molecules\-: \doxyref{\-Z\-Scatterer}{p.}{a00156}) there are as many positions as atoms.

\-A scatterer also has a few functions to display itself in 3\-D

\-This is an abstract base class.
